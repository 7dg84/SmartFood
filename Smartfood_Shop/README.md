# SmartFood Shop

Sistema de tienda para la aplicación SmartFood.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

La tienda estará disponible en `http://localhost:5175`

## Construcción

```bash
npm run build
```

## Configuración

Asegúrate de configurar la URL de la API en `src/config/api.ts`
