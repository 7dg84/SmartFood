# SmartFood Dashboard

Panel de administración para la aplicación SmartFood.

## Instalación

```bash
npm install
```

## Desarrollo

```bash
npm run dev
```

El dashboard estará disponible en `http://localhost:5174`

## Construcción

```bash
npm run build
```

## Configuración

Asegúrate de configurar la URL de la API en `src/config/api.ts`
